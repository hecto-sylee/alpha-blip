> 화면 ID는 `SCR-##`로 부여한다.

## 정보 구조 (IA)

```
앱
├─ 온보딩 (비로그인)
│   ├─ SCR-01 로그인 / 회원가입
│   └─ SCR-02 반려동물 프로필 등록
│
├─ 홈 / 산책 (S1)
│   ├─ SCR-10 홈 (산책 시작 전)
│   ├─ SCR-11 산책 지도 (산책 중)
│   ├─ SCR-12 상대 프로필 미리보기 (바텀시트)
│   ├─ SCR-13 요청 대기 / 결과
│   └─ SCR-14 매칭 세션 (함께 산책 중)
│
├─ 기록 (S3)
│   ├─ SCR-20 기록 작성 에디터
│   ├─ SCR-21 다이어리 (캘린더)
│   ├─ SCR-22 기록 상세
│   ├─ SCR-23 방 목록
│   ├─ SCR-24 방 생성
│   ├─ SCR-25 방 상세 (기록 타임라인)
│   └─ SCR-26 방 초대 / 방 코드
│
└─ 마이 / 설정 (공통)
    ├─ SCR-30 마이페이지
    ├─ SCR-31 반려동물 프로필 관리/수정
    └─ SCR-32 개인정보 보호 설정
```

### 전역 내비게이션 (탭바)

MVP 하단 탭 3개 제안:

| 탭 | 진입 화면 | 비고 |
|---|---|---|
| 산책 | SCR-10 홈 | 기본 진입 탭 |
| 기록 | SCR-21 다이어리 | 다이어리 + 방 |
| 마이 | SCR-30 마이페이지 | 프로필·설정 |

## 화면별 정의

### 온보딩

#### SCR-01. 로그인 / 회원가입
- **목적:** 계정 생성·인증
- **구성:** 소셜 로그인/이메일, 약관 동의
- **진입/이탈:** 앱 최초 진입 → 성공 시 SCR-02(신규) 또는 SCR-10(기존)

#### SCR-02. 반려동물 프로필 등록 (F-02)
- **목적:** 필수 프로필 입력
- **구성:** 이름·사진·견종·나이·성별·크기(필수), 성격 태그(최소 1), 선택 항목
- **규칙:** 필수 미입력 시 다음 비활성. 완료 → SCR-10
- **이탈:** 중도 이탈 시 입력값 임시 저장(권장)

---

### 홈 / 산책 (S1)

#### SCR-10. 홈 (산책 시작 전)
- **목적:** 산책 시작 진입점
- **구성:** 큰 [산책 시작] 버튼, 내 반려동물 요약, 최근 산책 요약(스탯), 위치 권한 상태
- **이탈:** [산책 시작] → SCR-11

#### SCR-11. 산책 지도 (산책 중) (F-01)
- **목적:** 근처 강아지 확인 + 산책 상태 유지
- **구성:** 지도, 내 위치 마커, 근처 강아지 아이콘(거리), 필터, [산책 종료] 버튼
- **상태:** 위치 권한 거부 시 안내/재시도, 근처 0명 시 빈 상태
- **이탈:** 아이콘 탭 → SCR-12 / [산책 종료] → SCR-20

#### SCR-12. 상대 프로필 미리보기 (바텀시트) (F-02/F-03)
- **목적:** 요청 전 상대 확인
- **구성:** 사진·이름·견종·크기·성격 태그·거리, [같이 산책하기] 버튼
- **이탈:** 요청 → SCR-13 / 닫기 → SCR-11

#### SCR-13. 요청 대기 / 결과 (F-03)
- **목적:** 요청 상태 표시
- **구성:** 대기 인디케이터, 취소 버튼, 결과(수락/거절/만료)
- **이탈:** 수락 → SCR-14 / 그 외 → SCR-11

#### SCR-14. 매칭 세션 (함께 산책 중) (F-04)
- **목적:** 함께 산책 상태 관리
- **구성:** 합류 안내(대략 위치+거리), 간단 메시지/이모지, 동행 시간, [산책 종료], 신고/차단, 매칭 취소
- **이탈:** [산책 종료] → 로그 저장(F-05) → SCR-20

---

### 기록 (S3)

#### SCR-20. 기록 작성 에디터 (F-10)
- **목적:** 산책 후 기록 작성
- **구성:** 미디어 추가(영상/사진), 텍스트·스티커·그림 에디터, 자동 정보(날짜/시간/거리/함께한 강아지), 공개 범위 선택기(일기/방 공유), [저장]
- **규칙:** 미디어 없이 텍스트만 저장 가능. 방 공유 선택 시 방 선택 UI
- **이탈:** 저장 → SCR-21 / 이탈 시 드래프트 저장(권장)

#### SCR-21. 다이어리 (캘린더) (F-10)
- **목적:** 기록 모아보기
- **구성:** 캘린더 뷰, 날짜별 기록 썸네일, 산책 스탯(누적 거리·횟수·연속 일수)
- **이탈:** 기록 탭 → SCR-22 / 방 진입 → SCR-23

#### SCR-22. 기록 상세 (F-10)
- **목적:** 단일 기록 열람
- **구성:** 미디어·텍스트, 산책 정보, 공개 범위 표시, 수정/삭제, 방 공유

#### SCR-23. 방 목록 (F-11)
- **목적:** 참여 중인 방 모아보기
- **구성:** 방 카드 리스트(이름·멤버·최근 기록), [방 만들기], [방 코드로 참여]
- **이탈:** 방 탭 → SCR-25 / 만들기 → SCR-24

#### SCR-24. 방 생성 (F-11)
- **목적:** 새 방 생성
- **구성:** 방 이름/반려동물 조합, 생성 → 초대(SCR-26)

#### SCR-25. 방 상세 (기록 타임라인) (F-11)
- **목적:** 방 공유 기록 열람
- **구성:** 기록 타임라인, 멤버 목록, [기록 올리기], [멤버 초대], 나가기
- **이탈:** 초대 → SCR-26

#### SCR-26. 방 초대 / 방 코드 (F-11)
- **목적:** 멤버 초대
- **구성:** 매칭 이력 상대 목록(초대), 방 코드 표시/공유, (친구 초대는 MVP 비활성)

---

### 마이 / 설정 (공통)

#### SCR-30. 마이페이지
- **목적:** 내 정보 허브
- **구성:** 내 반려동물 요약, 산책 스탯, 설정 진입, 로그아웃

#### SCR-31. 반려동물 프로필 관리/수정 (F-02)
- **목적:** 프로필 조회·수정
- **구성:** SCR-02와 동일 항목, 수정 저장

#### SCR-32. 개인정보 보호 설정 (F-09)
- **목적:** 위치·기록 공개 통제
- **구성:** 위치 공유, 대략적 위치 표시, 집 주변 노출 방지, 기록 공개 범위 기본값, 차단 목록
- **비활성(MVP):** 친구 알림·Pet Space·동네 활동 공개

## 미확정 / 디자인 검토 필요

- 하단 탭 3개 구성(산책/기록/마이) 확정 여부
- 산책 중(SCR-11/14) 화면 전환 방식 (풀스크린 지도 vs 카드)
- 기록 에디터(SCR-20) 편집 도구 범위 (스티커·그림 MVP 포함 수준)
- 방 목록을 기록 탭 하위로 둘지 별도 탭으로 분리할지
