# MVP 사용자 플로우

> 범위: **S1 산책 매칭 + S3 기록 + 공통**
> 전체 플로우는 [whole_planning/03_user_flow.md](../whole_planning/03_user_flow.md) 참고.
> 이 문서는 MVP 범위 내 플로우만 다룬다. (S2/S4/S5 플로우 제외)

---

## 전체 흐름 한눈에

```
[온보딩] 회원가입 → 반려동물 프로필 등록
   │
   ▼
[홈] 산책 시작 → 오늘의 퀘스트 확인/선택 (F-12)
   │
   ├─ (근처에 강아지 있음) ──▶ 매칭 산책 ─┐
   └─ (없음 / 매칭 안 함) ───▶ 혼자 산책 ─┤  (산책 중 2초 클립으로 퀘스트 미션 촬영)
   │                                       │
   ▼ ◀─────────────────────────────────────┘
[산책 종료] ──▶ 기록 남기기 (퀘스트 클립 + 일기 / 방 공유)
   │
   ▼
[재방문] 다이어리 확인 · 방 기록 쌓기 · 새 퀘스트
```

## 1. 온보딩 플로우 (공통)

```
회원가입 / 로그인
  → 보호자 기본 정보 입력
  → 반려동물 프로필 등록 (F-02, 필수)
       ├─ 이름·사진·견종·나이·성별·크기 (필수)
       └─ 성격 태그 최소 1개 (필수)
  → 위치 권한 안내
  → 홈 진입
```

- 반려동물 프로필 미등록 시 산책 시작 불가 → 등록 화면으로 유도

---

## 2. 매칭 산책 플로우 (S1 → S3)

```
홈에서 [산책 시작] (F-01)
  → 오늘의 퀘스트 확인/선택 (F-12, 선택)
  → 위치 권한 허용 → 지도 활성화
  → 근처 강아지 아이콘 확인
  → 아이콘 탭 → 프로필 미리보기 (F-02)
  → [같이 산책하기] 요청 (F-03)
  → 상대 수락 → 매칭 세션 시작 (F-04)
       ├─ 합류 안내 (대략 위치 + 거리)
       └─ 간단 메시지
  → 함께 산책 (퀘스트 미션 순간을 2초 클립으로 촬영)
  → [산책 종료]
  → 매칭 로그 자동 저장 (F-05)
  → ▶ 기록 남기기 화면 (F-10, 퀘스트 클립 자동 연결)
```

### 분기: 요청 결과

```
[같이 산책하기] 요청
  ├─ 수락 → 매칭 세션
  ├─ 거절 → 지도로 복귀 (부드러운 안내)
  ├─ 시간 만료 → 지도로 복귀
  └─ 취소 → 지도로 복귀
```

## 3. 혼자 산책 플로우 (S1 → S3)

매칭이 없어도 산책과 기록은 가능하다. (매칭 비종속 사용)

```
홈에서 [산책 시작]
  → 오늘의 퀘스트 확인/선택 (F-12, 선택)
  → 지도 활성화 (근처 없음 또는 매칭 안 함)
  → 혼자 산책 (퀘스트 미션 순간을 2초 클립으로 촬영)
  → [산책 종료]
  → ▶ 기록 남기기 화면 (F-10, 퀘스트 클립 자동 연결)
```

## 4. 기록 플로우 (S3)

### 4-1. 기록 작성 (F-10)

```
기록 남기기 화면 (산책 종료 후 자동 진입)
  → 오늘의 퀘스트 클립 확인 (F-12, 산책 중 촬영한 2초 클립이 자동 모임)
  → 클립/사진 추가 (선택)
  → 텍스트·스티커·그림 꾸미기 (선택)
  → 공개 범위 설정
       ├─ 일기 (나만 보기)
       └─ 방 공유 → 공유할 방 선택 (F-11)
  → 저장
  → 다이어리(캘린더)에 반영 (퀘스트 배지와 함께)
```

> 미디어 없이 텍스트만으로도 저장 가능
> 퀘스트는 기록 주제를 제공할 뿐 필수 아님 (무시하고 자유 기록 가능)
> MVP에서는 "동네 공유" 옵션 없음

### 4-1-1. 오늘의 퀘스트 (F-12)

```
산책 시작 또는 기록 진입 시
  → 오늘의 퀘스트 후보 3개 제시
  → 1개 선택 → 그날의 주제로 고정(lock)
  → 산책 미션 안내 ("지금 찍어볼 순간")
  → 산책 중 2초 클립으로 미션 촬영
  → 기록에 자동 연결
```

> 방(F-11)에서는 그날 첫 시작 멤버가 고른 퀘스트를 멤버 전체가 공유한다.

### 4-2. 다이어리 열람

```
다이어리 진입
  → 캘린더 뷰 (날짜별 기록)
  → 기록 상세 확인
  → 산책 스탯 확인 (누적 거리·횟수·연속 일수)
```

### 4-3. 방(Room) 공유 (F-11)

```
방 만들기
  → 방 이름/반려동물 조합 + 방 모드(산책 친구 / 가족) 지정
  → 멤버 초대
       ├─ 매칭 이력 1회 이상인 상대 초대
       └─ 참여 코드(6자리) 공유 (앱 외부 공유 가능)
  → 방 오늘의 퀘스트 공유 (F-12, 모드별 팩)
  → F-10 기록을 방에 업로드
  → 멤버가 기록/클립에 이모지 반응
  → 방 히스토리(타임라인) 확인
  → 새 기록 업로드 시 참여자 알림
```

```
참여 코드로 참여
  → 6자리 참여 코드 입력 (또는 딥링크)
  → 방 참여 (인원 초과/중복 참여 처리)
  → 방 기록 열람 / 내 기록 업로드 / 이모지 반응
```

## 5. 설정 플로우 (공통)

```
설정 진입
  → 위치 공유 / 대략적 위치 표시 / 집 주변 노출 방지
  → 기록 공개 범위 기본값 (일기 / 방 공유)
  → 차단 목록 관리
```
