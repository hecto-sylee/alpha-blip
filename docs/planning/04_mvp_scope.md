| 회원가입 / 로그인 | |

## MVP 검증 지표

| 지표 | 의미 |
|---|---|
| 산책 시작 버튼 클릭률 | 핵심 기능 진입률 |
| 위치 공유 허용률 | 서비스 사용 가능 사용자 비율 |
| 근처 강아지 프로필 조회율 | 탐색 의도 |
| 같이 산책하기 요청률 | 핵심 행동 전환율 |
| 요청 수락률 | 매칭 품질 |
| 매칭 성공률 | 전체 전환율 |
| 산책 로그 생성률 | 산책 완료율 |
| 친구 요청률 | 관계 형성 의도 |
| 재방문율 | 서비스 지속 사용 여부 |

---

## Phase별 기능 확장

| 단계 | 목표 | 핵심 기능 |
|---|---|---|
| **Phase 1. 웹 MVP** | 핵심 UX 검증 | 위치 권한, 산책 시작/종료, 근처 강아지 지도 표시, 프로필 확인, 산책 요청/수락, 매칭 로그 |
| **Phase 2. 앱 MVP** | 실제 산책 상황 검증 | 앱 기반 위치 수집, 푸시 알림, 친구 요청, 반복 매칭 기록, 산책 후 피드백, 기본 친밀도 |
| **Phase 3. 정식 서비스** | 위치 기반 P2P 산책 네트워크 구축 | 친구 시스템, 친밀도 시스템, 산책 호출, 그룹 산책, 커뮤니티, AI 추천, 고도화 개인정보 보호 |
