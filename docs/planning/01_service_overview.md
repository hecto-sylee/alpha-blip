# 서비스 개요

## 서비스 정의

산책 중인 반려동물과 반려인을 실시간으로 연결하는 위치 기반 산책 매칭 플랫폼

> 서비스명 미정

---

## 핵심 가치

| 가치 | 설명 |
|---|---|
| 탐색 시간 단축 | 지금 근처에서 함께 산책 가능한 반려견을 즉시 확인 |
| 성향 기반 연결 | 반려견 성격·산책 스타일 기반 매칭 |
| 네트워크 형성 | 동네 반려인 간 느슨한 커뮤니티 형성 |
| 관계 전환 | 일회성 만남 → 지속적 산책 친구 관계 |
| 반복 사용 유도 | 산책 기록·관계 데이터 기반 재방문 유도 |

---

## 문제 정의

**문제 상황:** 반려인은 매일 산책하지만, 지금 근처에서 함께 산책할 수 있는 반려견과 반려인을 찾기 어렵다.

**핵심 Pain Point:** 산책 친구 탐색의 비효율성

| 세부 Pain Point |
|---|
| 근처 산책 중인 반려견 정보 부재 |
| 산책 시간과 장소 조율의 번거로움 |
| 반려견 성향 기반 친구 탐색의 어려움 |
| 동네 반려인 네트워크 부족 |
| 우연한 만남에 의존하는 산책 친구 형성 |
| 산책은 반복 행동이지만 관계 형성으로 이어지기 어려움 |

---

## 타깃 사용자

**1차 타깃:** 도심 또는 주거 밀집 지역에서 반려견을 산책시키는 보호자

### 주요 사용자 상황

- 매일 또는 주기적으로 강아지 산책을 한다
- 산책 중 다른 강아지와 자연스럽게 만나고 싶다
- 우리 강아지와 성향이 맞는 친구를 찾고 싶다
- 동네 반려인과 느슨한 관계를 만들고 싶다
- 채팅으로 약속을 잡는 과정은 부담스럽다

### 초기 핵심 사용자

| 유형 |
|---|
| 소형견 또는 중형견 보호자 |
| 사회성이 있는 반려견 보호자 |
| 산책 빈도가 높은 보호자 |
| 동네 커뮤니티 형성 니즈가 있는 보호자 |
| 1인 가구 또는 젊은 반려인 |

---

## 핵심 컨셉

### 기존 vs 본 서비스 방식

| 기존 매칭 방식 | 본 서비스 방식 |
|---|---|
| 매칭 → 채팅 → 일정 조율 → 만남 | 산책 시작 → 근처 강아지 확인 → 프로필 확인 → 요청 → 수락 → 즉시 만남 → 로그 저장 → 친구 형성 |

**핵심 차별점:** 사전에 약속을 잡는 매칭이 아니라, 산책 중인 사용자를 실시간으로 연결하는 방식

---

## 서비스 핵심 문장

이 서비스는 단순한 펫 SNS가 아니라, 산책 중인 반려인들을 실시간으로 연결하고 반복 만남을 통해 동네 반려인 네트워크를 형성하는 **위치 기반 P2P 산책 플랫폼**이다.

처음에는 "지금 근처에서 함께 산책할 친구를 찾는 문제"를 해결하고, 이후에는 친구 시스템과 친밀도 시스템을 통해 지속적인 산책 관계를 만든다.
