# 사용자 핵심 플로우

---

## 신규 사용자 플로우

```
회원가입
  → 보호자 기본 정보 입력
  → 반려견 프로필 등록
  → 산책 스타일 설정
  → 위치 권한 허용
  → 산책 시작
```

---

## 실시간 산책 매칭 플로우

```
산책 시작
  → 지도에서 근처 강아지 확인
  → 프로필 확인
  → [같이 산책하기] 요청
  → 상대방 수락
  → 매칭 성사
  → 함께 산책
  → 산책 종료
  → 매칭 로그 저장
  → 친구 요청 (선택)
```

---

## 반복 사용 플로우

```
친구 목록 확인
  → 친밀도 높은 친구 확인
  → [같이 산책할래요] 알림 발송
  → 상대방 수락
  → 반복 산책
  → 친밀도 상승
  → 베스트 산책 메이트 전환
```
