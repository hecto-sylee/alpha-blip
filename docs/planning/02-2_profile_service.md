# 02-2. 프로필 서비스

> 관련 기능: F-02 (반려견 프로필)
> MVP 포함: ✅ Phase 1

---

## 서비스 개요

반려견의 성향과 기본 정보를 등록·관리하는 서비스.
같이 산책하기 전 상대방의 반려견을 파악하는 주요 수단이며, 매칭 필터링과 친밀도 계산의 기반 데이터가 된다.

---

## F-02. 반려견 프로필

**목적:** 같이 산책하기 전 반려견의 성향과 기본 정보를 확인한다.

### 프로필 항목

| 항목 | 필수 여부 |
|---|---|
| 이름 | 필수 |
| 사진 | 필수 |
| 견종 | 필수 |
| 나이 | 필수 |
| 성별 | 필수 |
| 크기 | 필수 |
| 중성화 여부 | 선택 |
| 성격 태그 | 필수 (최소 1개) |
| 사회성 | 선택 |
| 활동량 | 선택 |
| 산책 스타일 | 선택 |
| 선호 친구 유형 | 선택 |
| 주의사항 | 선택 |

### 성격 태그 목록 (예시)

`사람 좋아함` `강아지 좋아함` `소심함` `활발함` `천천히 걷기 선호` `뛰는 산책 선호` `대형견 무서워함` `짖음 있음`
