# 서비스 개요

## 서비스 미션

> **"내 반려동물이 더 풍요로운 하루를 보낼 수 있도록"**

반려동물의 행복을 중심에 두는 서비스.
강아지에게는 산책과 사회화가 그 답이고, 모든 반려동물에게는 기록과 커뮤니티가 그 답이다.

## 서비스 정의

산책 중인 반려동물과 반려인을 실시간으로 연결하는 위치 기반 산책 매칭 플랫폼이자,
반려동물의 일상을 기록하고 커뮤니티로 연결하는 반려동물 중심 라이프 플랫폼.

> 서비스명 미정

---

## 서비스 확장 구조

```
[레이어 1] 산책 매칭          →  강아지 전용 (핵심, 깊게)
[레이어 2] 활동 기록 / 다이어리 →  모든 반려동물 (pet-agnostic)
[레이어 3] 커뮤니티            →  동물 종류별 니치 커뮤니티로 분화
```

레이어 1은 강아지 사용자를 앱으로 끌어오는 핵심 훅.
레이어 2·3은 강아지 외 반려인들도 앱을 사용하게 되는 확장 경로.

## 핵심 가치

| 가치 | 설명 |
|---|---|
| 탐색 시간 단축 | 지금 근처에서 함께 산책 가능한 반려견을 즉시 확인 |
| 성향 기반 연결 | 반려견 성격·산책 스타일 기반 매칭 |
| 네트워크 형성 | 동네 반려인 간 느슨한 커뮤니티 형성 |
| 관계 전환 | 일회성 만남 → 지속적 산책 친구 관계 |
| 반복 사용 유도 | 산책 기록·관계 데이터 기반 재방문 유도 |
| 반려동물이 주인공 | 반려동물 자신의 공간(Pet Space)을 통해 기록과 관계를 쌓음 |
| 내 동물을 아는 커뮤니티 | 프로필·산책 이력이 쌓인 상태에서 커뮤니티로 자연스럽게 연결 |

## 문제 정의

**문제 상황:** 반려인은 매일 산책하지만, 지금 근처에서 함께 산책할 수 있는 반려견과 반려인을 찾기 어렵다.
또한 산책 기록이 쌓이지 않아 산책 습관 형성과 앱 재방문이 이어지지 않는다.
기존 커뮤니티는 내 동물을 모르는 상태에서 글을 쓸 때마다 동물 정보를 반복 소개해야 한다.

**핵심 Pain Point:**

| 세부 Pain Point |
|---|
| 근처 산책 중인 반려견 정보 부재 |
| 산책 시간과 장소 조율의 번거로움 |
| 반려견 성향 기반 친구 탐색의 어려움 |
| 동네 반려인 네트워크 부족 |
| 우연한 만남에 의존하는 산책 친구 형성 |
| 산책은 반복 행동이지만 관계 형성으로 이어지기 어려움 |
| 산책 기록이 쌓이지 않아 루틴 형성이 안 됨 |
| 기존 커뮤니티는 내 동물 맥락 없이 반복 소개가 필요 |
| 온라인 커뮤니티에서 같은 동네 반려인과 연결할 방법이 없음 |

## 타깃 사용자

### 1차 타깃: 강아지 보호자 (산책 매칭)

도심 또는 주거 밀집 지역에서 반려견을 산책시키는 보호자

| 유형 |
|---|
| 소형견 또는 중형견 보호자 |
| 사회성이 있는 반려견 보호자 |
| 산책 빈도가 높은 보호자 |
| 동네 커뮤니티 형성 니즈가 있는 보호자 |
| 1인 가구 또는 젊은 반려인 |

### 2차 타깃: 모든 반려동물 보호자 (기록·커뮤니티)

강아지 외 반려동물을 키우며 기록을 남기고 커뮤니티에 연결되고 싶은 보호자

| 반려동물 | 주요 니즈 |
|---|---|
| 고양이 | 품종별 건강 정보, 실내 활동 기록 |
| 토끼 | 식이/질병 정보, 운동공간 공유 |
| 앵무새/조류 | 훈련·언어 발달, 품종별 특성 |
| 파충류 | 사육 환경, 먹이 정보 |
| 소동물 | 수명·질병, 케어 정보 |

## 핵심 컨셉

### 기존 vs 본 서비스 방식

| 기존 매칭 방식 | 본 서비스 방식 |
|---|---|
| 매칭 → 채팅 → 일정 조율 → 만남 | 산책 시작 → 근처 강아지 확인 → 프로필 확인 → 요청 → 수락 → 즉시 만남 → 기록 → 친구 형성 |

**핵심 차별점:** 사전에 약속을 잡는 매칭이 아니라, 산책 중인 사용자를 실시간으로 연결하는 방식

### 반려동물이 주인공인 공간 (Pet Space)

```
인스타:  주인(계정 주체) → 반려동물 콘텐츠 업로드 → 주인의 팔로워
이 앱:   반려동물(공간 주체) → 주인이 기록을 채워줌 → 다른 반려동물·반려인
```

- 각 반려동물이 자신의 공간(Pet Space)을 가짐
- 산책 기록, 매칭 이력이 반려동물 관점에서 쌓임
- 매칭으로 친해진 다른 반려동물의 공간을 방문할 수 있음

### 내 동물을 아는 커뮤니티

기존 커뮤니티는 글을 쓸 때마다 내 동물을 소개해야 한다.
이 서비스는 프로필, 산책 기록, 매칭 이력이 이미 쌓인 상태에서 커뮤니티로 연결된다.
커뮤니티가 추가 기능이 아니라, 기록과 관계의 자연스러운 다음 단계가 된다.

## 기존 서비스와의 차별화

| 기존 서비스 | 페인 포인트 | 이 서비스의 대안 |
|---|---|---|
| 네이버 카페 | 글마다 내 동물 반복 소개, 낡은 UX, 지역성 약함 | 프로필 기반, 모바일 최적화, 동네 기반 연결 |
| 당근마켓 동네생활 | 반려동물 맥락 없음, 관계 연결 약함 | 실제 산책 이력 기반 신뢰 관계 |
| 인스타그램 펫 계정 | 주인이 주인공, 반려동물은 소재 | 반려동물이 공간의 주인공 |
| 동물 전문 앱 (포동, 핏펫 등) | 기록과 커뮤니티 분리 | 기록 → 커뮤니티 공유가 한 흐름 |

## 서비스 핵심 문장

이 서비스는 단순한 펫 SNS가 아니라, 산책 중인 반려인들을 실시간으로 연결하고 반복 만남을 통해 동네 반려인 네트워크를 형성하는 **위치 기반 P2P 산책 플랫폼**이다.

핵심 기능인 산책 매칭 위에, 반려동물이 주인공인 기록 공간(Pet Space)과 동네·종별 커뮤니티가 레이어로 쌓여,
강아지 보호자뿐 아니라 모든 반려인이 자신의 동물과 함께하는 일상을 기록하고 연결되는 플랫폼으로 성장한다.
