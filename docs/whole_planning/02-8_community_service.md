# 02-8. 커뮤니티 서비스

> 소속 시스템: **S5. 커뮤니티**
> 관련 기능: F-13 (동네 커뮤니티), F-14 (니치 커뮤니티)
> Phase: F-13 Phase 3 / F-14 Phase 4

---

## 시스템 개요

S5 커뮤니티는 S3 기록(F-10)이 콘텐츠 공급원이고, S1 매칭 이력이 있는 사용자 기반으로 신뢰 커뮤니티를 형성한다.
"내 동물을 아는 커뮤니티" — F-02 프로필과 산책 이력이 쌓인 상태에서 커뮤니티로 자연스럽게 연결되는 것이 기존 커뮤니티(네이버 카페 등)와의 핵심 차별점.

### 기존 커뮤니티와의 차별화

| 기존 서비스 페인 포인트 | S5 커뮤니티의 대안 |
|---|---|
| 글마다 내 동물 반복 소개 | F-02 프로필 기반 — 내 동물을 이미 아는 상태 |
| 지역성 약함 (전국 단위) | 동네 기반 커뮤니티 (F-13) |
| 온라인→오프라인 연결 없음 | 동네 피드에서 매칭 요청 또는 친구 요청으로 연결 |
| 텍스트 중심, 낡은 UX | F-10 영상/다이어리 기반 기록 UX |
| 기록과 커뮤니티 분리 | F-10 기록 → F-13 공유가 한 흐름 |
| 신뢰 기반 관계 없음 | 실제 산책 이력이 있는 사용자 기반 |

---

## F-13. 동네 커뮤니티 (열린 공유 레이어)

**목적:** F-10 기록을 동네 반려인과 공유하고, 지역 반려인 네트워크를 형성한다.

> F-10 없이는 피드를 채울 콘텐츠가 없다. F-13은 F-10의 열린 공유 목적지다.

### 주요 기능

| 기능 | 설명 |
|---|---|
| 동네 피드 | 같은 동네 반려동물 기록을 피드 형태로 확인 |
| 동네 반려인 발견 | 자주 산책하는 근처 반려인 탐색 |
| 게시글 작성 | 동네 공개 기록 또는 자유 게시글 |
| 댓글 / 반응 | 커뮤니티 상호작용 |
| 모임 제안 | 동네 그룹 산책 또는 모임 공지 |

### 피드 콘텐츠 유형

| 유형 | 출처 |
|---|---|
| 산책 기록 공유 | F-10에서 동네 공개로 설정한 기록 |
| 매칭 완료 기록 | 매칭 후 공유 동의 시 피드에 노출 |
| 자유 게시글 | 반려동물 관련 자유 글 |
| 모임 제안 | 그룹 산책 또는 모임 공지 |

### 기획 원칙

- 동네 기반 게시글은 정확한 위치를 노출하지 않는다
- 초반 피드 공백 해소: 산책 기록 자동 공유 옵션 (사용자 선택)
- 피드에서 발견한 반려인에게 S1 매칭 요청 또는 S2 친구 요청 가능

---

## F-14. 니치 커뮤니티 (종별 커뮤니티)

**목적:** 반려동물 종별 관심사 기반 커뮤니티를 제공한다.

### 종별 커뮤니티

| 반려동물 | 커뮤니티 주요 콘텐츠 | 산책 매칭 확장 가능성 |
|---|---|---|
| 고양이 | 품종별 건강 정보, 실내 환경 구성 | 하네스 산책 유저 한정 |
| 토끼 | 식이/질병 정보, 운동공간 공유 | 야외 놀이 모임 (매우 니치) |
| 앵무새/조류 | 훈련·언어 발달, 품종별 특성 | 공원 외출 모임 가능성 |
| 파충류 | 사육 환경, 먹이 정보 | 기록 레이어(S3)만 유효 |
| 소동물 | 수명·질병, 케어 정보 | 기록 레이어(S3)만 유효 |

### 확장 원칙

- F-13 동네 커뮤니티 활성화 이후 단계적으로 확장
- F-02 프로필에 등록된 반려동물 종 기반으로 자동 커뮤니티 추천
- S3 F-10 기록이 니치 커뮤니티의 콘텐츠 생산 기반
