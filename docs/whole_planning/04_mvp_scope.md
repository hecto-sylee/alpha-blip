# MVP 범위 및 Phase 계획

> 시스템 구분(S1~S5, 공통)은 [02_feature_spec.md](./02_feature_spec.md) 참고.

---

## MVP 구현 범위

MVP는 **S1(산책 매칭) + S3(기록) + 공통** 세 영역만 구현한다.
S2(소셜/관계), S4(Pet Space), S5(커뮤니티)는 MVP 이후로 미룬다.

| 시스템 | MVP 포함 | 비고 |
|---|---|---|
| 공통 | ✅ | F-02 프로필, F-09 개인정보 보호 설정 |
| S1 산책 매칭 | ✅ | F-01, F-03, F-04, F-05 |
| S3 기록 | ✅ | F-10 다이어리, F-11 방(Room) |
| S2 소셜/관계 | ❌ | 친구·친밀도·호출 (MVP 이후) |
| S4 Pet Space | ❌ | MVP 이후 |
| S5 커뮤니티 | ❌ | 동네·니치 커뮤니티 (MVP 이후) |

## MVP 범위 내 제약 (타 시스템 의존 항목 비활성)

S2·S4·S5를 구현하지 않으므로, 해당 시스템에 의존하는 일부 옵션은 MVP에서 비활성화한다.

| 기능 | MVP 제약 | 사유 |
|---|---|---|
| F-10 공개 범위 | **일기 / 방 공유**만 제공 (동네 공유 제외) | 동네 공유 = S5 커뮤니티 미구현 |
| F-11 방 멤버 초대 | **매칭 이력 초대 + 방 코드 공유**만 제공 (친구 초대 제외) | 친구 초대 = S2 친구 시스템 미구현 |
| F-09 설정 항목 | S1·S3 관련 항목만 (위치, 기록 공개 범위) | 친구별 알림·Pet Space 공개·동네 활동 공개는 해당 시스템 미구현 |
| 산책 종료 후 친구 요청 제안 | 미노출 | S2 미구현 |

## MVP 검증 지표

### S1 산책 매칭

| 지표 | 의미 |
|---|---|
| 산책 시작 버튼 클릭률 | 핵심 기능 진입률 |
| 위치 공유 허용률 | 서비스 사용 가능 사용자 비율 |
| 근처 강아지 프로필 조회율 | 탐색 의도 |
| 같이 산책하기 요청률 | 핵심 행동 전환율 |
| 요청 수락률 | 매칭 품질 |
| 매칭 성공률 | 전체 전환율 |

### S3 기록

| 지표 | 의미 |
|---|---|
| 산책 후 기록 작성률 | 루틴 형성 핵심 지표 |
| 혼자 산책 기록 비율 | 매칭 비종속 사용 여부 |
| 방(Room) 생성·업로드율 | 공유 레이어 작동 여부 |
| 재방문율 | 서비스 지속 사용 여부 |

## MVP 이후 확장 계획

| 단계 | 추가 시스템 | 핵심 기능 |
|---|---|---|
| **MVP** | 공통 + S1 + S3 | 프로필, 산책 매칭, 산책 기록 다이어리, 방(Room) 공유 |
| **확장 1** | S2 소셜/관계 | 친구 시스템, 친밀도, 산책 호출 → 기록 공개 범위에 친구 초대 활성화 |
| **확장 2** | S4 Pet Space | 반려동물 중심 공간 (기존 S1·S3 데이터로 자동 구성) |
| **확장 3** | S5 커뮤니티 | 동네 커뮤니티 → F-10 동네 공유 활성화 → 종별 니치 커뮤니티 분화 |

---

## 레이어 확장 구조

```
MVP:
  [공통]     프로필 · 개인정보 설정
  [레이어 1] 산책 매칭 (S1)  →  강아지 전용, 핵심 검증
  [레이어 2] 활동 기록 (S3)  →  산책 루틴 형성 + 방(Room) 공유

확장기:
  [레이어 1] 산책 매칭 (S1)
  [레이어 2] 활동 기록 (S3) + 관계 (S2) + Pet Space (S4)
  [레이어 3] 커뮤니티 (S5)  →  동네 → 종별 니치 커뮤니티로 분화
```
