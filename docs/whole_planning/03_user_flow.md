# 사용자 핵심 플로우

---

## 신규 사용자 플로우

```
회원가입
  → 보호자 기본 정보 입력
  → 반려동물 프로필 등록
  → 산책 스타일 설정 (강아지인 경우)
  → 위치 권한 허용 (강아지인 경우)
  → 홈 / Pet Space 진입
```

---

## 실시간 산책 매칭 플로우

```
산책 시작
  → 지도에서 근처 강아지 확인
  → 프로필 확인
  → [같이 산책하기] 요청
  → 상대방 수락
  → 매칭 성사
  → 함께 산책
  → 산책 종료
  → 기록 남기기 화면 진입 (F-10)
  → 기록 작성 / 꾸미기 (선택)
  → 공개 범위 설정 (일기 / 방 공유 / 동네 공유)
  → 저장 완료
  → 친구 요청 제안 (선택)
```

---

## 혼자 산책 기록 플로우

```
산책 시작
  → 지도 활성화 (매칭 없음)
  → 산책 완료 버튼 클릭
  → 기록 남기기 화면 진입 (F-10)
  → 기록 작성 / 꾸미기 (선택)
  → 공개 범위 설정 (일기 / 방 공유 / 동네 공유)
  → 저장 완료
```

---

## 반복 사용 플로우

```
친구 목록 확인
  → 친밀도 높은 친구 확인
  → [같이 산책할래요] 알림 발송
  → 상대방 수락
  → 반복 산책
  → 기록 남기기 (F-10)
    → 방(Room)에 공유 (F-11)
  → 친밀도 상승
  → 베스트 산책 메이트 전환
```

---

## Pet Space 플로우

```
내 Pet Space 진입
  → 내 반려동물의 산책 기록 피드 확인
  → 공간 꾸미기 (배경, 소품, 스티커 등)
  → 친구 반려동물의 Pet Space 방문
  → 방문 반응 남기기
```

---

## 방(Room) 플로우

```
친구 관계 성립 (1회 이상 매칭)
  → 방 생성 (반려견 조합 또는 그룹 지정)
  → 산책 기록 업로드 (개인 다이어리에서 방으로 공유)
  → 방 히스토리 확인
  → 그룹 산책 초대 (친밀도 3단계)
```

---

## 동네 커뮤니티 플로우

```
동네 피드 진입
  → 같은 동네 반려동물 산책 기록 피드 탐색
  → 게시글 확인 / 반응
  → 개인 기록 동네 공개 설정으로 피드에 올리기
  → 동네 반려인 발견 → 매칭 요청 또는 친구 요청
```
