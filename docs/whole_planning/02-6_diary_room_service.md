# 02-6. 산책 기록 다이어리 & 방(Room) 서비스

> 소속 시스템: **S3. 기록**
> 관련 기능: F-10 (산책 기록 다이어리), F-11 (방 기반 공유 기록)
> Phase: F-10 Phase 2 / F-11 Phase 3

---

## 시스템 개요

S3 기록 시스템은 산책을 기록하고 공유하는 레이어.
F-10 다이어리가 콘텐츠 생산 레이어이고, F-11 방(Room)은 그 위에 올라오는 닫힌 공유 레이어다.
매칭 유무와 무관하게 산책 후 기록을 남길 수 있으며, S5 동네 커뮤니티로의 공유도 F-10 기록에서 출발한다.

```
F-10 산책 기록 다이어리 (콘텐츠 생산)
  └─ 공개 범위 설정
       ├─ 일기 (나만 보기)
       ├─ F-11 방(Room) 공유  →  특정 멤버와 공유 (닫힌 공유)
       └─ 동네 공유           →  S5 동네 커뮤니티 피드로 연결
```

---

## F-10. 산책 기록 다이어리 (콘텐츠 생산 레이어)

**목적:** 산책 완료 후 기록을 남겨 루틴 형성과 앱 재방문을 유도한다. F-11(방 공유)·S5(동네 커뮤니티)의 콘텐츠 공급원 역할을 한다.

> 매칭 유무와 무관하게 산책 완료 기록은 항상 남길 수 있다.
> 매칭이 있었다면 상대방 반려동물 정보와 함께 기록된다.

### 기록 기본 포맷

기본 포맷은 **영상** (산책 중 촬영하거나, 사진들로 자동 구성).
영상/사진 위에 텍스트, 스티커, 그림을 얹어 꾸밀 수 있다 (다꾸 감성).
미디어 없이 텍스트만으로도 기록 가능 (진입 장벽 최소화).

### 기록 진입 플로우

```
산책 종료 버튼 클릭 (S1 F-04)
  → 기록 남기기 화면 자동 진입
  → 영상/사진 업로드 또는 촬영 (선택)
  → 텍스트·스티커·그림 등으로 꾸미기 (선택)
  → 공개 범위 설정
       ├─ 일기 (나만 보기)
       ├─ F-11 방(Room) 공유
       └─ 동네 공유 (S5 F-13)
  → 저장
```

### 기록 포함 정보

| 항목 | 저장 방식 |
|---|---|
| 산책 날짜 / 시간 / 거리 | 자동 (S1 F-05 데이터) |
| 함께한 반려동물 (매칭이 있었을 경우) | 자동 (S1 F-05 데이터) |
| 영상 / 사진 | 사용자 업로드 또는 촬영 |
| 텍스트, 스티커, 그림 | 사용자 입력 |
| 공개 범위 | 사용자 선택 |

### 주요 UI

| 요소 | 설명 |
|---|---|
| 기록 꾸미기 에디터 | 영상/사진 위에 텍스트·스티커 추가 |
| 캘린더 뷰 | 날짜별 산책 기록 확인 |
| 산책 스탯 | 누적 거리, 산책 횟수, 연속 일수 |
| 공개 범위 선택기 | 일기 / 방 공유 / 동네 공유 |

### 기획 원칙

- 기록을 남기는 행위가 산책 루틴의 마무리가 되도록 UX 설계 (열린 루프 닫기)
- 다꾸 감성 + 운동 기록 앱 스탯 구조
- 미디어 없이 텍스트만으로도 기록 가능

## F-11. 방(Room) 기반 공유 기록 (닫힌 공유 레이어)

**목적:** F-10에서 생산된 기록을 특정 멤버와 공유하는 닫힌 공간을 만든다.

> F-10 없이는 방에 올릴 콘텐츠가 없다.

### 방 참여 방법

| 방법 | 설명 |
|---|---|
| 매칭 이력 기반 초대 | 1회 이상 매칭 기록이 있는 상대를 초대 |
| 친구 초대 | 이미 친구 추가된 상대를 초대 (S2 F-06) |
| 방 코드 공유 | 방 코드를 직접 공유해 참여 (앱 외부 공유 가능) |

### 주요 기능

| 기능 | 설명 |
|---|---|
| 방 생성 | 누구나 생성 가능, 반려동물 조합 또는 그룹 이름 지정 |
| 멤버 초대 | 매칭 이력 / 친구 / 방 코드 세 가지 방식 |
| 기록 업로드 | F-10 기록을 방에 공유 |
| 방 히스토리 | 함께한 산책 기록 아카이브 타임라인 |
| 방 참여 관리 | 초대 / 나가기 |
| 방 알림 | 새 기록 업로드 시 참여자 알림 |

### 활용 시나리오

- 자주 같이 산책하는 친구 반려동물과의 기록 모아보기
- 그룹 산책 히스토리 아카이브
- 방 코드를 통해 앱 밖에서 아는 견주를 온보딩
