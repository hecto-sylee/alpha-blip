# 02-2. 프로필 서비스

> 관련 기능: F-02 (반려견 프로필), F-12 (Pet Space)
> MVP 포함: ✅ Phase 1 (프로필) / Phase 3 (Pet Space)

---

## 서비스 개요

반려동물의 성향과 기본 정보를 등록·관리하는 서비스.
같이 산책하기 전 상대방의 반려동물을 파악하는 주요 수단이며, 매칭 필터링과 친밀도 계산의 기반 데이터가 된다.
Phase 3에서는 각 반려동물의 Pet Space로 확장되어, 반려동물이 주인공인 개인 공간을 갖게 된다.

---

## F-02. 반려견 프로필

**목적:** 같이 산책하기 전 반려견의 성향과 기본 정보를 확인한다.

### 프로필 항목

| 항목 | 필수 여부 |
|---|---|
| 이름 | 필수 |
| 사진 | 필수 |
| 견종 | 필수 |
| 나이 | 필수 |
| 성별 | 필수 |
| 크기 | 필수 |
| 중성화 여부 | 선택 |
| 성격 태그 | 필수 (최소 1개) |
| 사회성 | 선택 |
| 활동량 | 선택 |
| 산책 스타일 | 선택 |
| 선호 친구 유형 | 선택 |
| 주의사항 | 선택 |

### 성격 태그 목록 (예시)

`사람 좋아함` `강아지 좋아함` `소심함` `활발함` `천천히 걷기 선호` `뛰는 산책 선호` `대형견 무서워함` `짖음 있음`

## F-12. Pet Space (Phase 3)

**목적:** 반려동물이 자신의 디지털 공간을 갖고, 기록과 관계가 반려동물 중심으로 쌓이도록 한다.

### Pet Space 구성

| 구성 요소 | 설명 |
|---|---|
| 피드 | 산책 기록, 매칭 기록이 반려동물 관점에서 쌓임 |
| 공간 꾸미기 | 주인이 반려동물의 방/집을 꾸며줌 (배경, 소품, 스티커 등) |
| 방문 | 매칭으로 친해진 다른 반려동물의 공간 방문 가능 |
| 반려동물 자기소개 | 의인화 톤으로 작성된 반려동물의 소개 |

### 의인화 UX

| 일반적인 반려동물 앱 | Pet Space 의인화 UX |
|---|---|
| "보미, 말티즈, 3살" | "안녕, 나는 보미야. 느린 산책이 제일 좋아." |
| 주인이 정보를 입력하는 폼 | 반려동물의 목소리로 채워지는 공간 |
| 기록 = 주인의 일기 | 기록 = 보미의 오늘 하루 |

의인화는 전체 UX의 톤앤매너로 자연스럽게 스며들어야 한다. 억지스러운 의인화는 오히려 어색하게 느껴지므로 강도 조절이 중요하다.
