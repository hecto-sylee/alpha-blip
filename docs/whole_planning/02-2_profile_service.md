# 02-2. 프로필 서비스

> 소속 시스템: **공통 (S1 산책 매칭 + S4 Pet Space)**
> 관련 기능: F-02 (반려동물 프로필)
> Phase: ✅ Phase 1

---

## 시스템 개요

F-02 반려동물 프로필은 전체 서비스의 공통 기반 데이터.
S1(산책 매칭)에서는 지도 아이콘·매칭 필터·상대 확인의 기반이 되고,
S4(Pet Space)에서는 반려동물 공간의 정체성 데이터가 된다.

| 시스템 | F-02가 쓰이는 방식 |
|---|---|
| S1 산책 매칭 | 지도 아이콘, 매칭 필터, 요청 전 상대 확인 |
| S2 소셜/관계 | 친밀도 계산 기반 (성격·선호 유형 활용) |
| S3 기록 | 산책 기록에 반려동물 정보 자동 포함 |
| S4 Pet Space | Pet Space의 정체성, 의인화 소개의 원천 데이터 |
| S5 커뮤니티 | 커뮤니티 글쓴이 맥락 (내 동물을 아는 커뮤니티) |

---

## F-02. 반려동물 프로필

**목적:** 반려동물의 성향과 기본 정보를 등록·관리한다.

### 프로필 항목

| 항목 | 필수 여부 |
|---|---|
| 이름 | 필수 |
| 사진 | 필수 |
| 종 / 견종 | 필수 |
| 나이 | 필수 |
| 성별 | 필수 |
| 크기 | 필수 |
| 중성화 여부 | 선택 |
| 성격 태그 | 필수 (최소 1개) |
| 사회성 | 선택 |
| 활동량 | 선택 |
| 산책 스타일 | 선택 |
| 선호 친구 유형 | 선택 |
| 주의사항 | 선택 |

### 성격 태그 목록 (예시)

`사람 좋아함` `강아지 좋아함` `소심함` `활발함` `천천히 걷기 선호` `뛰는 산책 선호` `대형견 무서워함` `짖음 있음`
