# 02-7. Pet Space 서비스

> 관련 기능: F-12 (Pet Space)
> MVP 포함: Phase 3

---

## 서비스 개요

반려동물이 자신의 디지털 공간을 갖는 서비스.
인스타그램에서 주인이 주인공이고 반려동물이 소재인 것과 달리,
이 앱에서는 반려동물이 공간의 주인공이 되어 기록과 관계가 반려동물 중심으로 쌓인다.

```
인스타:  주인(계정 주체) → 반려동물 콘텐츠 업로드 → 주인의 팔로워
이 앱:   반려동물(공간 주체) → 주인이 기록을 채워줌 → 다른 반려동물·반려인
```

싸이월드 미니홈피를 반려동물이 소유하는 감각에 가깝다.

---

## F-12. Pet Space

**목적:** 반려동물이 자신의 디지털 공간을 갖고, 기록과 관계가 반려동물 중심으로 쌓이도록 한다.

### Pet Space 구성 요소

| 구성 요소 | 설명 |
|---|---|
| 피드 | 산책 기록, 매칭 기록이 반려동물 관점에서 쌓임 |
| 공간 꾸미기 | 주인이 반려동물의 방/집을 꾸며줌 (배경, 소품, 스티커 등) |
| 반려동물 자기소개 | 의인화 톤으로 작성된 소개 |
| 방문 | 매칭으로 친해진 다른 반려동물의 공간 방문 |
| 방문 반응 | 방문한 공간에 반응 남기기 |

### 실세계-디지털 연결 구조

```
실세계 레이어:  산책 매칭 → 실제로 만남
디지털 레이어:  Pet Space → 친구 집에 디지털로 놀러감
```

산책에서 맺어진 관계가 디지털 공간에서도 이어지는 구조.

### 의인화 UX

| 일반적인 반려동물 앱 | Pet Space 의인화 UX |
|---|---|
| "보미, 말티즈, 3살" | "안녕, 나는 보미야. 느린 산책이 제일 좋아." |
| 주인이 정보를 입력하는 폼 | 반려동물의 목소리로 채워지는 공간 |
| 기록 = 주인의 일기 | 기록 = 보미의 오늘 하루 |

의인화는 전체 UX의 톤앤매너로 자연스럽게 스며들어야 한다.
억지스러운 의인화는 오히려 유치하게 느껴지므로 강도 조절이 중요하다.

### 공간 꾸미기 아이템 분류

| 분류 | 예시 |
|---|---|
| 배경 | 공원, 집, 하늘, 계절 테마 |
| 가구·소품 | 방석, 장난감, 밥그릇 |
| 스티커 | 반려동물 종별 스티커, 감정 표현 |
| 시즌 아이템 | 명절, 기념일 한정 아이템 |

### 기획 원칙

- 반려동물을 의인화해서 바라보는 성향의 유저에게 강하게 작동
- 의인화 톤이 전체 UX에서 일관되게 유지되어야 함
- Pet Space 꾸미기가 산책 매칭이라는 메인 기능을 희석시키지 않도록 레이어 분리 필요
- Pet Space는 기록(F-10)의 자연스러운 집이 되도록 설계
