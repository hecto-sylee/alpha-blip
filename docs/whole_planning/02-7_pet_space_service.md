# 02-7. Pet Space 서비스

> 소속 시스템: **S4. Pet Space**
> 관련 기능: F-12 (Pet Space)
> Phase: Phase 3

---

## 시스템 개요

S4 Pet Space는 반려동물이 주인공인 디지털 공간.
다른 시스템에서 쌓인 데이터(F-02 프로필, S1 매칭 이력, S3 기록)가 Pet Space를 채워준다.
사용자가 직접 만들어가는 것이 아니라, 산책을 하고 기록을 남기는 것만으로 자연스럽게 채워지는 공간이다.

```
F-02 프로필       →  Pet Space 정체성 (이름, 성격, 소개)
S1 매칭 이력      →  Pet Space 피드 (만난 친구 기록)
S3 F-10 기록     →  Pet Space 피드 (산책 일상 기록)
S2 친구 관계      →  다른 반려동물 Pet Space 방문 가능
```

---

## F-12. Pet Space

**목적:** 반려동물이 자신의 디지털 공간을 갖고, 기록과 관계가 반려동물 중심으로 쌓이도록 한다.

### Pet Space 구성

| 구성 요소 | 채워지는 데이터 출처 |
|---|---|
| 반려동물 자기소개 | F-02 프로필 (의인화 톤으로 변환) |
| 피드 | S3 F-10 산책 기록, S1 F-05 매칭 이력 |
| 공간 꾸미기 | 주인이 직접 꾸밈 (배경, 소품, 스티커) |
| 방문 | S2 친구 관계의 다른 반려동물 Pet Space |

### 의인화 UX

| 일반적인 반려동물 앱 | Pet Space 의인화 UX |
|---|---|
| "보미, 말티즈, 3살" | "안녕, 나는 보미야. 느린 산책이 제일 좋아." |
| 주인이 정보를 입력하는 폼 | 반려동물의 목소리로 채워지는 공간 |
| 기록 = 주인의 일기 | 기록 = 보미의 오늘 하루 |

의인화는 전체 UX의 톤앤매너로 자연스럽게 스며들어야 한다.
억지스러운 의인화는 오히려 어색하므로 강도 조절이 중요하다.

### 실세계-디지털 연결 구조

```
실세계:   산책 매칭(S1) → 실제로 만남
디지털:   Pet Space(S4) → 친구 집에 디지털로 놀러감
```

### 공간 꾸미기 아이템

| 분류 | 예시 |
|---|---|
| 배경 | 공원, 집, 하늘, 계절 테마 |
| 가구·소품 | 방석, 장난감, 밥그릇 |
| 스티커 | 반려동물 종별 스티커, 감정 표현 |
| 시즌 아이템 | 명절, 기념일 한정 아이템 |

### 기획 원칙

- Pet Space는 별도로 "만들어가는" 공간이 아니라, 산책·기록이 쌓이면서 자연스럽게 채워지는 공간
- Pet Space 꾸미기가 산책 매칭(S1)이라는 메인 기능을 희석시키지 않도록 레이어 분리 필요
