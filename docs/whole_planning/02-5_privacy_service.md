# 02-5. 개인정보/설정 서비스

> 소속 시스템: **공통 (전 시스템)**
> 관련 기능: F-09 (개인정보 보호 설정)
> Phase: ✅ Phase 1 (기본) / Phase 3 (기록·커뮤니티 설정)

---

## 시스템 개요

F-09 개인정보 보호 설정은 전 시스템에 걸쳐 적용되는 공통 설정.
위치(S1), 관계 알림(S2), 기록 공개 범위(S3), Pet Space 공개(S4), 커뮤니티 활동(S5) 각각의 공개 수준을 사용자가 직접 통제한다.
위치 기반 서비스 특성상 신뢰 형성의 핵심 요소이며, MVP 단계부터 기본 설정을 제공한다.

---

## F-09. 개인정보 보호 설정

**목적:** 사용자가 각 시스템에서의 공개 범위를 직접 통제한다.

### 설정 항목 (시스템별)

| 설정 | 설명 | 관련 시스템 |
|---|---|---|
| 위치 공유 ON/OFF | 산책 중에만 위치 공유 | S1 |
| 산책 종료 시 자동 비공개 | 종료 즉시 지도에서 제거 | S1 |
| 대략적 위치 표시 | 정확한 좌표 대신 반경 내 표시 | S1 |
| 집 주변 위치 노출 방지 | 홈 반경 내 위치 비공개 처리 | S1 |
| 마지막 위치 자동 만료 | 일정 시간 후 위치 데이터 삭제 | S1 |
| 차단 / 신고 | 특정 사용자 차단 | S1, S2 |
| 친구별 알림 허용 설정 | 특정 친구 알림만 수신 | S2 |
| 특정 시간대 알림 제한 | 야간 등 알림 차단 시간 설정 | S2 |
| 기록 공개 범위 설정 | 일기 / 방(Room) 공유 / 동네 공유 선택 | S3 |
| Pet Space 공개 여부 | Pet Space를 친구에게만 / 전체 공개 / 비공개 | S4 |
| 동네 커뮤니티 활동 공개 설정 | 동네 피드 활동 공개/비공개 | S5 |
