# 스냅챌(SnapChal) 사업성 분석 조사

> 작성일: 2026-05-26
> 목적: 사업성 중심 발표 PPT 제작을 위한 시장 조사 및 BM 설계

---

## 1. 서비스 개요

- **서비스명**: 스냅챌 (SnapChal)
- **한 줄 정의**: 친구·커플이 하루의 퀘스트 주제에 맞춰 매시간 2초 영상을 찍고, 하루 끝에 셋로그처럼 모아보는 폐쇄형 소셜 앱
- **플랫폼**: Android MVP
- **핵심 가설**: 퀘스트가 있으면 셋로그형 기록 경험이 더 재미있어지는가

## 3. 경쟁 서비스 분석

### 3-1. 셋로그 (Setlog) — 가장 직접적인 경쟁자

| 항목 | 내용 |
|---|---|
| 형태 | iOS 우선 → 2026년 4월 Android 베타 |
| 핵심 기능 | 1시간 알림 → 2~4초 촬영 → 하루 끝 자동 합성 |
| 그룹 규모 | 2~12명 폐쇄형, 초대 코드 |
| 앱스토어 성과 | 2026년 4월 **전체 1위** (구글 제미나이, 스레드 제침) |
| Android 출시 성과 | **5일 만에 50만 다운로드** |
| ARR | **111억 원** (2025년 5월 기준) |
| 월 결제액 | **10억 원** |
| 수익성 | 현금흐름 기준 **흑자 전환** |
| 누적 투자 | 46.5억 원 (소프트뱅크벤처스, 알토스벤처스) |
| 글로벌 비중 | 전체 사용자의 **35%가 해외** |
| 수익화 | 구독 기반 추정 |

**셋로그의 한계 (스냅챌의 기회)**
- 퀘스트/테마 기반 미션 **없음** → 찍을 이유가 매번 사용자 스스로 만들어야 함
- iOS 우선으로 Android 사용자 초기 소외
- BeReal과 유사한 단일 기능 의존 리스크

### 3-2. BeReal — 글로벌 흥망 반면교사

| 시기 | MAU |
|---|---|
| 2022년 8월 (피크) | **7,350만 명** |
| 2023년 3월 | 3,300만 명 (피크 대비 55% 하락) |
| 2025년 초 | **1,600만 명** (피크 대비 **78% 하락**) |
| 2024년 | 500M 유로에 Voodoo에 매각 |

**하락 원인 (스냅챌 설계 시 반드시 피해야 할 것)**
1. **신기함 소진**: 핵심 경험 자체가 반복되면 흥미 소실
2. **단일 기능**: 인스타그램 등이 복제하면 무방비
3. **참여 동기 부재**: 왜 찍어야 하는지 이유가 없어짐
4. **수익화 지연**: 3년간 광고 없이 운영 → 매각

**수익화 (2025년 뒤늦게 도입)**
- 인피드 광고 + 브랜드 파트너십
- 2025년 광고 수익 $3,000만 달성 (기존 $0에서 급성장)

### 3-3. 인스타그램 친친 (Close Friends)

- 2024~2025년 피드·릴스까지 친친 확장
- Z세대의 **38%가 친한 친구 전용 부계정** 운영 (Pew Research)
- Z세대의 **71%가 공개 게시물로 인한 대인 갈등 경험** → 프라이빗 공유 선호 가속화
- 수익화 없음 (Meta 광고 생태계에 포함)

| 서비스 | 영상 | 폐쇄형 | 퀘스트/미션 | 친구·커플 | 수익화 |

## 4. 폐쇄형 소셜 앱 트렌드

**핵심 드라이버**

| 요인 | 내용 |
|---|---|
| 디지털 피로감 | 공개 피드의 과도한 콘텐츠 소비에 지침 |
| 퍼포먼스 압박 해소 | 좋아요·팔로워 수에 집착 없는 소통 선호 |
| 프라이버시 우려 | TikTok 금지, 개인정보 유출 이슈 |
| 진정성 추구 | 연출 없는 일상 수요 증가 |
| 소규모 연결감 | 많은 팔로워보다 깊은 관계 선호 |

- 전 세계 소셜 미디어 이용자의 **약 절반이 커뮤니티 기반 플랫폼 이용 시간 증가** 계획 (Sprout Social 2026)
- Discord, Reddit 그룹, Instagram Broadcast Channels 등 폐쇄형 커뮤니티로 이동 중

## 5. 비즈니스 모델 (확정)

> 레퍼런스: 스노우 앱 + 셋로그 구독 구조 + 일본 스티커 사진(purikura) 감성

### 수익 4단계 구조

| 티어 | 내용 |
|---|---|
| **무료** | 기본 이모지 (하트, 반짝 등 기본 몇 가지), 기본화질 저장 |
| **광고 보기 (Rewarded Ad)** | 광고 1회 시청 → 특별 스티커 1개 일회성 사용 |
| **프리미엄 구독** | 광고 없음 + 스티커 무제한 (스노우 스타일/purikura 이펙트) + 고화질 저장 |
| **인앱 구매 (IAP)** | 시즌 퀘스트 팩 (크리스마스, 여름 여행, 커플 기념일 등) |

### 프리미엄 구독 세부 혜택
- 광고 없음
- 스티커/이펙트 무제한 (일본 스티커 사진 느낌 이펙트 포함)
- 고화질 저장 및 다운로드
- (추후 고려) AI 하이라이트 자동 편집, 추억 달력 리플레이

### 수익화 단계별 로드맵

| 단계 | 시점 | 내용 |
|---|---|---|
| Phase 1 | MVP | 완전 무료, 바이럴 성장 집중 |
| Phase 2 | MVP 안정화 후 | 프리미엄 구독 도입 + Rewarded Ad |
| Phase 3 | 사용자 확보 후 | 시즌 퀘스트 팩 IAP + 브랜드 스폰서 퀘스트 |

## 6. 차별화 포지셔닝

> **핵심 한 줄**: "찍을 이유가 매일 생기는 유일한 폐쇄형 숏폼 소셜 앱"

- BeReal이 실패한 이유 = 찍을 이유 부재 → 스냅챌이 퀘스트로 해결
- 셋로그가 검증한 시장 → 스냅챌이 퀘스트로 차별화
- 스노우가 검증한 스티커 수익화 → 스냅챌이 소셜 기록에 결합
